
# Theme for Potes

Theme is a WordPress block based theme. The following CSS classes can be used for blocks:

## Patterns

* `potes-header` Colorfull header with image
* `potes-large-section-header` Centered header with a big heading and sub text line
* `potes-person-bio` Block with photo, name, job title and short bio
* `potes-section-image-test` Block containing image and text

## Classes

`motto` can be used on paragraphs, large font, can be used for slogans/motto's/line below header

`potes-person-bios` can be used on rows to group multiple pattern groups of type potes-person-bio

`potes-file-list` can be used on list, use strong for the filename and no formatting for extra metadata such as `PDF`

`container` can be used on groups, gives some padding on mobile layouts, set a width in the properties
